# 🚀 NGXR Counter

> The NGXR Counter is a simple counter app built using Angular's state management library, NGRX.

![Counter](/screen/home.gif)

## ✨ Features
- 📟 Displays the current count on screen
- ➕ Increment
- ➖ Decrement
- 🔄 Reset the counter

## 🛠️ Technologies
- 🅰️ Angular
- ⚡ NgRX
- 🧠 TypeScript
- 🎨 SCSS
- 🧱 BEM (Block Element Modifier)
